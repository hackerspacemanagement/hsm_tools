HsmTools
========

This plugin adds tool management to [Hackerspace
Management](https://github.com/rrix/hsm).

Installation
------------
Add the following to your HSM Gemfile:

```@ruby
gem 'hsm_tools', :source => "https://github.com/rrix/hsm_tools"
```

Run:

```@bash
$ bundle
$ rails g hsm_tools:install
$ rake db:migrate
```
